# 🕵️ Investigation Support System

An AI-powered surveillance system for real-time person detection, face recognition, multi-object tracking, evidence capture, and SQLite-based detection logging.

## 🚀 Features

- Real-time Person Detection (YOLOv8)
- Face Recognition
- Multi-Object Tracking
- Evidence Capture
- SQLite Detection Log
- Streamlit Dashboard (In Progress)

## 🛠️ Tech Stack

- Python
- OpenCV
- YOLOv8
- Face Recognition (dlib)
- SQLite
- Streamlit

python app.py
